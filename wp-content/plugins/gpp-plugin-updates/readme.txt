== Description ==

Provides one-click updates for all Graph Paper Press plugins and extensions.

== Installation ==

1. Upload the whole `gpp-plugin-updates` directory to the `/wp-content/plugins/` directory
1. Activate the 'GPP Plugin Updates' plugin through the 'Plugins' menu in WordPress
1. In WordPress, click Dashboard -> Updates to update your plugins and extensions.
